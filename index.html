What is HTML

HTML (Hypertext Markup Language) is the standard markup language for creating web pages and web applications.

What is Semantic HTML

Semantic HTML refers to the use of HTML tags that convey meaning beyond their visual presentation, helping to structure and define the content of a webpage in a meaningful way.

Common Semantic Tags

  • <header>
  • <nav>
  • <main>
  • <footer>
  • <article>
  • <section>
  • <aside>
  • <figure>
  • <figcaption>
